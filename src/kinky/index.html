kinky.tgz is my entire IRAF package with scripts. The package is broken into various sub packages.

To install this package......
  1. getthe file kinky.tgz (a gzipped tart file).  (shift-click the link)
  2. cd ~/iraf (ie.  go to the directory where you start iraf from)
  3. gzcat kinky.tgz | tar xvf - (this will create a subdirectory called kinky)
  4. edit you login.cl file and add the following lines (after the first goup of set commands is good)
Stuff to add to your login.cl file
set kinky = home$kinky/
task kinky.pkg = kinky$kinky.cl

 

kinky tasks

color
The IRAF color package. Useful for making color versions of images (if you have input image in more than one filter)
edhead
Task to modify CFH12k headers to work with the enclosed version of mscred
fitsutil
Package of tasks designed to modify fits multi-extension-format (MEF) files.
fkbo
Tasks that are useful for dealing with sequences of images, intended to assist in finding moving objects
flip
Flips the internal structure of CFH12k single chip fits files to make them self consistent. This doesn't change the physical storage of the images.  If you want to re-orient the images use ewflip in the fkbo package.
gcs
package of my globular cluster system scripts.
mscred
The October-18 1999 release of the mscred package. Works correctly with edhead and cfh12k. (other version of mscred may work also... I don't know).
