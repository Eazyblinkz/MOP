MOP

The Moving Object Pipeline: discovery & tracking of trans-Neptunian objects

Download this project as a .zip file Download this project as a tar.gz file

MOP is a software pipeline designed to detect trans-Neptunian objects in CFHT MEGAPRIME data. The driver programs are mostly in Python 2.7 with the analysis components in Fortran and IRAF.

Mop maintained by OSSOS
